# 🤖 GFS-X Sim-to-Real RL Agent

Этот проект посвящен управлению реальным гусеничным роботом XiaoR Geek GFS-X с использованием обучения с подкреплением (Reinforcement Learning) в архитектуре **Hardware-In-the-Loop (HIL)**. 

Обучение агента (PPO + LSTM) происходит в симуляторе Unity ML-Agents, после чего обученная модель (`.onnx`) управляет физическим роботом через ROS-мост, принимая данные с реальных датчиков в режиме реального времени.

## 🛠 Архитектура системы

Проект разделен на две основные части:
1. **Unity Environment (`/UnityProject`)**: Цифровой испытательный полигон. Содержит 3D-модель робота, скрипты `VirtualSensors`, конфигурации ML-Agents и логику начисления наград (Curriculum Learning).
2. **ROS / Raspberry Pi (`/RaspberryPi_Nodes`)**: Аппаратный шлюз на базе RPi 4B. Скрипты отвечают за чтение аппаратных ИК-датчиков, обработку компьютерного зрения (YOLO) и управление ШИМ-сигналами дифференциального привода.

## 🚀 Ключевые особенности
* **Асинхронное HIL-управление:** Отказ от пространственной синхронизации цифрового двойника ради устранения проблемы счисления пути (Dead Reckoning Drift).
* **Sensor Fusion:** Объединение данных ультразвукового дальномера, ИК-сенсоров и компьютерного зрения в единый вектор наблюдений размерностью 10.
* **Deadzone Mapping:** Программная адаптация линейных выходов нейросети к физическому трению покоя коллекторных двигателей.
